Inventory SQL chatbot.
TO RUN THIS CODE ON UR LAPTOP:
1-Open a terminal and run:
  git clone https://github.com/YOUR_USERNAME/inventory_SQL_Chatbot.git
  cd inventory_SQL_Chatbot

2-virtual environment:
  python -m venv .venv
  source .venv/bin/activate or .venv\Scripts\activate

3-install :
  pip install -r requirements.txt

4- create ur env file and add your Groq API key.
5- run the project: python main.py.
AFTER RUNNING THE PROJECT TYPE (exit) TO STOP IT.
